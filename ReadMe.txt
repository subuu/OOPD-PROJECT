1. Install SSMS setup from drive link.
2. After successful installation open the application.
3. Some general commands.
-create database OOPDProjectDB. Press enter.
-create table dummytb(username nvarchar(max),userpassword nvarchar (100),userphone nvarchar(20)). Press enter.
-insert into dummytb values('name','name12345',123456789). Press enter.
4.After successful installation of DB.
5.Create java project in ide(I used eclipse).
6.Download and unzip the file: sqljdbc_9.4.0.0_enu.zip
7.Link jre 8 from sqljdbc_9.4.0.0_enu > enu > jre8 in java project by 
(left click project > build path > configure build path >libraries > add external jars > jre8)
8. open sql server manager console from start menu and make TCP/IP enabled under network configuration on left side panel.
